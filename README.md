# Lotto Project

로또 번호 생성 및 관리 시스템

## Project Structure

```
lotto-project/
├── frontend/    # Next.js 프론트엔드
└── backend/     # NestJS 백엔드
```

## Tech Stack

### Frontend
- Next.js 16
- TypeScript
- Tailwind CSS
- ESLint

### Backend
- NestJS
- TypeScript
- Node.js

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Frontend will run on http://localhost:3000

### Backend
```bash
cd backend
npm install
npm run start:dev
```

Backend will run on http://localhost:3000

## Development

This is a monorepo structure. Each project can be run independently.
